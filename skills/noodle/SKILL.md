# Noodle

Meta-skill for configuring and repairing Noodle.

Use this skill to:
- Diagnose `noodle.toml` issues.
- Repair missing adapter scripts/paths.
- Explain runtime files under `.noodle/`.
- Verify `noodle start`, `noodle status`, and `noodle skills list` behavior.

Core principle: fix root causes with minimal, durable changes.
