# Share What You Know

When delegating work, include analysis you've already done. If you've identified file overlap between tasks, dependency ordering, or environment constraints — put it in the prompt. Don't make the delegate spend turns re-deriving conclusions you already have.

Every piece of analysis withheld is a turn wasted rediscovering it. The cost of a longer prompt is far less than the cost of redundant exploration.

This is [[principles/foundational-thinking|sequencing]] applied to delegation — analysis already done is infrastructure that every downstream turn benefits from. Front-load it.

Applies to: director→manager prompts, manager→worker prompts, any delegation boundary.
