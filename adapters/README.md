# Adapters

This is not part of Noodle itself. These are the Noodle project's own backlog adapters — the scripts that Noodle uses to manage its own `brain/todos.md` backlog when running Noodle on itself. The default for new projects is `todos.md` (see `defaults/adapters/`).

Other projects using Noodle write their own adapters (conventionally at `adapters/`).
